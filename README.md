\# Smart Attendance System Using Face Recognition



\## Overview



A Smart Attendance System that automatically marks attendance using Face Recognition technology. The system detects faces from a webcam feed, identifies registered users, and stores attendance records in a CSV file.



\## Features



\* Face Detection

\* Face Recognition

\* Automatic Attendance Marking

\* CSV Attendance Storage

\* Real-Time Recognition

\* OpenCV Integration



\## Technologies Used



\* Python

\* OpenCV

\* NumPy

\* Face Recognition

\* Pandas
